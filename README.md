## Start

This is starter project develop from nestjs https://nestjs.com/

## Installation

```bash
$ yarn install

```

## Running database

```bash
$ docker-compose up -d
$ use pgadmin localhost:5050 create server and database

```

## Running the app

```bash
$ yarn dev

```

## Example env

```bash
$ PS_HOST=localhost
$ PS_PORT=root
$ PS_USER=root
$ PS_DATABASE=booking

```
